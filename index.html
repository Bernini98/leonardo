For part of my university placement year, I worked at Leonard on a Big Data platform for processing radar information. My role involved developing a Python front-end and connecting it to the Hadoop framework such as representing and manipulating data in HBase

Python was used to develop the front-end and use Thrift to speak to HBase

Django is a web framework utilised so employees could access the big data platform through a browser and utilise validation functionality

The Bootstrap css framework was used to produce a clear, presentable front-end

The Hadoop framework was used to build the Big Data platform. I became familiar with a variety of tools for processing Big Data such as Spark and HBase

I connected the front-end with the Active Directory of the company so engineers could use their Windows login to use the system

Atlassian's Jira was used to delicate tasks for team sprints and connect with Bitbucket

Confluence was used to document in detail my work with the front-end system to ensure someone new to the platform could continue the work
